Coding Contest Reminder

A React-based web application to help programmers never miss coding contests. It aggregates upcoming contests from multiple platforms and provides an interactive interface with filtering, search, dark/light theme, and modal details.

Features

View Upcoming Contests
See all upcoming contests from popular platforms like LeetCode, CodeChef, HackerRank, Codeforces, AtCoder, Google Kick Start, Codewars.

Search & Filter

Search contests by name

Filter by platform: LeetCode, CodeChef, HackerRank, Codeforces, AtCoder, Kick Start, Codewars

Filter by time: Today, This Week, This Month

Contest Details Modal
Click a contest to view detailed information including date, time, duration, and platform.

Dark/Light Theme Toggle
Easily switch between themes; the app remembers your preference.

Responsive Design
Works on desktop, tablet, and mobile devices.

Technologies Used

Frontend: React, HTML5, CSS3, JavaScript

Icons & UI: Font Awesome

State Management: React Hooks (useState, useEffect)

Styling: External CSS file (style.css)

Installation & Setup

Clone the repository

git clone https://github.com/Udayteja10/coding-contest-reminder.git
cd coding-contest-reminder


Install dependencies

npm install


Start the development server

npm start


Open the app in your browser at http://localhost:3000
